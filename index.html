  • Disqover

    This project was completed as part of a group. Disqover is a social media application that seeks to connect users who have similar music and movie interests. The Application is a full-stack application using MEAN Stack (Mongo, Express, AngularJS, and NodeJS). The application's features include the ability to search an external api for movies or music, like them, and add them to your collection. You can also view other people's disqover pages. You can also click "Follow" to follow someone's disqover.
