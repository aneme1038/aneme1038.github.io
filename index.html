Star Wars Trivia

A FrontEnd Static Trivia Game Application based on the Star Wars Universe. The Application is built using HTML, CSS, Javascript, jQuery, and AJAX. This project was my first ever fully coded frontend application.

Tweed-ly

A Full-Stack Multi-page CRUD application that features a user based collection of Men's Tweed Suits to share between users. The application is built using embedded Javascript with NodeJS/Express and MongoDB for the backend database.

Disqover

This project was completed as part of a group. Disqover is a social media application that seeks to connect users who have similar music and movie interests. The Application is a full-stack application using MEAN Stack (Mongo, Express, AngularJS, and NodeJS). The application's features include the ability to search an external api for movies or music, like them, and add them to your collection. You can also view other people's disqover pages. You can also click "Follow" to follow someone's disqover.

Gameplayce

Gameplayce is a full-stack CRUD application using Ruby, Rails, and ReactJS. The application was built as part of a group and features the ability to post reviews for video games that are searchable through an external API.

Wed-ly

Wed-Ly is a project currently under development but visible to the public during its development process. Currently, the main purpose behind the application is to learn a completely new language, framework, or methodology. I chose VueJS as the language to learn for this specific project. The project currently features a basic checklist where a person can view and delete entries. The aim is to get full CRUD capability and continue to add features as is needed to improve the application
